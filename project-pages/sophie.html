Portfolio de Allal Contactez-Moi
Sophie Bluel
Mon expérience en tant que développeur front-end m'a permis de travailler sur un projet passionnant consistant à créer une page web dynamique avec JavaScript. J'ai été sollicité par l'agence ArchiWebos pour rejoindre leur équipe en tant que développeur front-end et participer à la conception du site portfolio d'une architecte d'intérieur.
Détails du projet
Ce projet s'est avéré être une expérience enrichissante en tant que développeur front-end pour l'agence ArchiWebos. L'équipe, composée de 50 membres, m'a confié la tâche de participer au développement d'une page web dynamique pour le portfolio d'une architecte d'intérieur. Après une réunion d'équipe informative, j'ai reçu un e-mail de Charlotte, la cheffe de projet, contenant les détails du projet, y compris le design Figma, le code back-end pour la gestion des données, le code front-end existant avec une version statique de la page de l'architecte, ainsi qu'un tableau Kanban détaillant les tâches qui me sont attribuées. Mon rôle principal consistait à développer trois éléments clés : la page de présentation des travaux de l'architecte, la page de connexion de l'administrateur du site (le client), et une modale permettant d'uploader de nouveaux médias. J'ai ainsi eu l'opportunité de travailler sur un projet dynamique et d'apprendre à manipuler le DOM avec JavaScript pour créer des fonctionnalités interactives.
Mon aventure avec ArchiWebos a débuté par une réunion d'équipe où j'ai pris des notes pour comprendre pleinement les attentes du projet. Par la suite, Charlotte m'a fourni toutes les ressources nécessaires, y compris les maquettes Figma pour le design et les codes sources pour le développement. J'ai été chargé de créer la page de présentation des travaux de l'architecte en utilisant le code HTML fourni, de développer la page de connexion du client à partir de zéro, et de créer une modale pour l'ajout de nouveaux médias. Mon apprentissage et ma mise en pratique des compétences en JavaScript ont été essentiels pour gérer les événements utilisateurs, manipuler les éléments du DOM, et gérer la récupération des données via des formulaires. Au terme de ce projet, j'ai acquis une expérience précieuse dans le développement web interactif avec JavaScript, tout en contribuant à l'expansion du portfolio de l'architecte d'intérieur.
