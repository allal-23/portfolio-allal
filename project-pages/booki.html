Booki
Ce projet consiste à créer la page d'accueil d'une agence de voyage en utilisant HTML et CSS. Il s'agit d'une tâche d'intégration où le développeur doit travailler en collaboration avec l'UI designer et la CTO pour concrétiser les maquettes et spécifications du site.
Détails du projet
Le projet consiste à créer la page d'accueil d'une agence de voyage en utilisant HTML et CSS. Cette mission d'intégration, initiée au sein de la start-up Booki, implique la transformation des maquettes conçues par l'UI designer en une interface web parfaitement fonctionnelle et responsive. En étroite collaboration avec Sarah, la CTO de l'entreprise, et Loïc, le designer UI, le développeur est responsable de traduire ces maquettes en code. Le projet démarre avec une structure de fichiers préparée par l'équipe, comprenant un fichier "index.html" pour le HTML, un dossier "css" contenant un fichier "style.css" pour le CSS, ainsi qu'un dossier "images" avec toutes les ressources visuelles nécessaires.
Le processus d'intégration est guidé par un souci de précision technique et une attention aux détails. Une note de synthèse réunit les spécifications et contraintes techniques, fournissant un cadre pour le projet. Le développeur est encouragé à éviter l'utilisation du code d'exemple trouvé sur Figma, à moins d'une parfaite compréhension. Une fois l'intégration terminée, l'ensemble du projet, y compris le code HTML, le CSS, et les images, est soumis sous forme d'un dossier ZIP. La soutenance finale, structurée en trois phases, implique une présentation des livrables, une discussion sur les choix techniques, et un débriefing. L'évaluation portera sur la capacité du développeur à créer une interface responsive avec HTML et CSS, ainsi que sur la précision de l'intégration conformément à la maquette et aux spécifications techniques. Ce projet offre une opportunité exceptionnelle de perfectionner les compétences de développement web tout en collaborant étroitement au sein d'une équipe multidisciplinaire.
